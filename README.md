This is a sandbox [Next.js](https://nextjs.org/) project to try new technologies and libraries. It should also become a kind of portfolio in the future.

## Used technologies and libraries

- next.js
- mongodb
- next-auth.js
- bcryptjs
- tailwind
- puppeteer
- nprogress
- ajax
